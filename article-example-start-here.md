# Markdown 模板示例

欢迎使用这份模板来撰写你的文章。Front Matter 会在上传时自动解析，生成标题、摘要、标签和日期。

## 写作指引

1. 在顶部补充准确的 Front Matter 信息。
2. 使用二级标题拆分段落，保持结构清晰。
3. 需要插图时，可在本地先记下占位说明，再在其他工具中处理图片。

## 代码示例

```js
export function greet(name) {
  return `Hello, ${name}!`;
}
```
